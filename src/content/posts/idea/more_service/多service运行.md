在多数情况下，通常会打开一个项目，里面有多个项目需要运行，这里教大家怎么开启多个窗口来查看项目运行状态，最终效果如下。
![img.png](img.png)
点击**view -> Tool Windows-> Services**
![img_1.png](img_1.png)
点击**services**之后，会在下面出现这个页面。
![img_2.png](img_2.png)
点击**Add Service -> Run Configuration Type**，选择**SpringBoot**
![img_3.png](img_3.png)
之后直接运行项目，就会实现最终效果。
![img_4.png](img_4.png)
